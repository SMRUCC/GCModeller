#### Compound
The kegg compound entry collection data in this pathway.
 (可以通过这个代谢物的列表得到可以出现在当前的这个代谢途径之中的所有的非酶促反应过程，
 将整个基因组里面的化合物合并起来则可以得到整个细胞内可能存在的非酶促反应过程)
