Where a ``3-mer`` means 3 consecutive DNA bases ranging from ``AAA`` to ``TTT``. 
 Each element of the ``1 x 64`` vector ``yi=(yi1, yi2, ... , yij, ... yi64)`` 
 represents the count Of its corresponding ``3-mer`` in the given sequence xi.
