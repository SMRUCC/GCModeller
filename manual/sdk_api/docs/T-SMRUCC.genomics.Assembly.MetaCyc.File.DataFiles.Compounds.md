# Compounds
_namespace: [SMRUCC.genomics.Assembly.MetaCyc.File.DataFiles](N-SMRUCC.genomics.Assembly.MetaCyc.File.DataFiles.html)_

> 
>  对于Compounds表而言，由于其包含的对象仅为小分子的代谢物，故而大分子的蛋白质分子不会出现在此列表之中
