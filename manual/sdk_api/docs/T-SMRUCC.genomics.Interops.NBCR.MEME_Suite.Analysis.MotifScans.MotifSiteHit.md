# MotifSiteHit
_namespace: [SMRUCC.genomics.Interops.NBCR.MEME_Suite.Analysis.MotifScans](N-SMRUCC.genomics.Interops.NBCR.MEME_Suite.Analysis.MotifScans.html)_

通过MAST程序扫描RegPrecise数据库里面的Motif Fasta所得到的结果




### Properties

#### RegPrecise
在RegPrecise数据库之中所匹配上的调控位点的trace信息：locus_tag:position
#### source
这个属性是记录了这个motif位点的来源信息，这个是用于Csv文档的，Xml文档会被忽略掉
