# iPfamPPI
_namespace: [SMRUCC.genomics.Analysis.ProteinTools.Interactions.iPfamPPI](N-SMRUCC.genomics.Analysis.ProteinTools.Interactions.iPfamPPI.html)_

比较基因组学里面的直系同源只是用于判断两个蛋白质在其他的物种里面同样也具有相互作用的可能性的高低，
 互作关系则需要从实验或者数据库之中获取。iPfam数据库则是用于注释互作关系的数据源，之后通过直系同源
 就可以简单的推断出需要进行注释的基因组里面的蛋白质互作网路了
