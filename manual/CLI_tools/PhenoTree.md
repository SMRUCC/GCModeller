<h3 id="/Cluster.Enrichment"> 1. /Cluster.Enrichment</h3>

```bash
PhenoTree /Cluster.Enrichment /in <partitions.json> /go.anno <proteins.go.annos.csv> [/go.brief <go_brief.csv> /out <out.DIR>]
```
