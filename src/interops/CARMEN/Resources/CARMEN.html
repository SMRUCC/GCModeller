KGML-based model reconstruction of metabolic pathways


After a CARMEN analysis run has finished, you can directly download various SBML files (Level 2 Version 1 and Version 4, Level 3 Version 1). Subsequently, the SBML file can be visualized by SBML-compliant tools.

Additionally, a SVG (Scalable Vector Graphics) representation of the metabolic reconstruction is rendered on the fly, displaying the selected metabolic pathways. The SVG image can be downloaded or reviewed in most recent browsers directly or via the freely available Adobe SVG plugin.


Please choose one organism of the left side and a multitude of KEGG pathways of the right side. Start the pathway reconstruction via the corresponding button!
