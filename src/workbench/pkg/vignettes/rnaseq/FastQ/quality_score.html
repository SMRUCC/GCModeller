quality_score {FastQ} R Documentation

In FASTQ files, quality scores are encoded into a compact form,

Description

which uses only 1 byte per quality value. In this encoding, the
quality score is represented as the character with an ASCII
code equal to its value + 33.

Usage

quality_score(q);

Arguments

q

should be one or more FastQ sequence data

env

[as Environment]

Details

Authors

rnaseq

Value

the quality score data of each FastQ sequence data.
