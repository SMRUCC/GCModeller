filter {geneExpression} R Documentation

Filter the geneID rows

Description

Usage

filter(HTS, geneId,
    exclude = FALSE);

Arguments

HTS

A gene expression matrix object. [as Matrix]

geneId

A character vector for run the matrix feature row filter. [as string]

exclude

matrix a subset of the data matrix excepts the input `geneId` features or just make a subset which just contains the input `geneId` features. [as boolean]

A new expression matrix object that consist with gene feature rows subset from the original matrix input.
