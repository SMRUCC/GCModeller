load.expr {geneExpression} R Documentation

load an expressin matrix data

Description

Usage

load.expr(file,
    exclude.samples = NULL,
    rm.ZERO = FALSE,
    makeNames = FALSE);

Arguments

file

the file path or the file stream data of the target expression matrix table file, or the expression data frame object

exclude.samples

will removes some sample column data from the expression matrix which is specificed by this parameter value. [as string]

Details

the table file format that handled by this function could be a csv table file or tsv table file.

Authors

phenotype_kit

Value

a HTS data matrix of samples in column and gene features in row

clr value class

  • Matrix

Examples

 load.expr(file = "./rawdata.csv")
