write.expr_matrix {geneExpression} R Documentation

write the gene expression data matrix file

Description

Usage

write.expr_matrix(expr, file,
    id = "geneID",
    binary = FALSE);

Arguments

expr

The gene expression matrix object

file

The file path to a csv matrix file that used
for export the given expr matrix data.

id

The string content inside the first cell

binary

write matrix data in binary data format? default value
is False means write matrix as csv matrix file.
