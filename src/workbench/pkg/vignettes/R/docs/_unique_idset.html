_unique_idset {GCModeller} R Documentation

helper function for makes the given id set unique

Description

Usage

_unique_idset(id);

Arguments

id

a character vector of the object id, duplicated value may contains inside.

Details

this function andalso will removes the id which is in empty literal value, example as: empty string, NA string, NULL string, etc.

Authors

xieguigang

Value

This function has no value returns.
