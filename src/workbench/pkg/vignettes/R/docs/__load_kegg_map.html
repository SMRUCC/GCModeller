__load_kegg_map {GCModeller} R Documentation

this function try to unify the kegg map load method

Description

Usage

__load_kegg_map(
    kegg.maps = NULL,
    raw.maps = FALSE);

Arguments

kegg.maps

the kegg map data source, could be a zip archive file that contains the kegg maps data message, a directory path that contains the kegg maps xml files. default parameter value is null, means use the internal resource data for run the downstream analysis.

Details

Authors

This function has no value returns.
