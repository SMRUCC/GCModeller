.readZipStream {GCModeller} R Documentation

Read zip stream data

Description

this function open the given zip archive file and then load the first file as data stream object if the zip entry name is missing.

Usage

.readZipStream(zipfile,
    entryName = NULL);

Arguments

zipfile

the file path of the target zip file to read data stream

entryName

the zip entry name.

Details

Authors

xieguigang

Value

A data steam object that read from the given zip archive file.
