Make motif site search in parallel

Description

Usage

scan_motifs(db, seqs,
    identities.cutoff = 0.8,
    minW = 0.85,
    top = 3,
    permutation = 2500,
    workdir = './',
    n.threads = 8);

Arguments

db

the file path to the motif PWM database

seqs

the file path to the TSS upstream site for make motif site matches
