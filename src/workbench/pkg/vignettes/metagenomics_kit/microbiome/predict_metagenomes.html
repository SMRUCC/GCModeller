predict_metagenomes {microbiome} R Documentation

creates the final metagenome functional predictions. It

Description

multiplies each normalized OTU abundance by each predicted
functional trait abundance to produce a table of functions
(rows) by samples (columns).

Usage

predict_metagenomes(PICRUSt, table);

Arguments

table

should be a merged OTU dataframe object, that should be in format like: 1. the colnames should be the sample name, and the column field value is the relative abundance value of each otu in each sample 2. the rows in this dataframe should be the otu expression value across samples the GCModeller internal Matrix is also avaiable for this parameter.
