build PICRUSt binary database file

Description

Usage

build.PICRUSt_db(ggtax, copyNumbers.16s, ko.13.5.precalculated, save);

Arguments

ggtax

A helper table gg138_99.gg.tax for make OTU id mapping to taxonomy information. [as otu_taxonomy]

ko.13.5.precalculated

file connection to the ko_13_5_precalculated.tab. [as Stream]

save

the file connection for save the compiled PICRUSt binary database file. [as Stream]

copyNumbers.16s

a list of the 16s RNA copy number, [#OTUIDs => 16SrRNA_Count]. [as list]

Details
