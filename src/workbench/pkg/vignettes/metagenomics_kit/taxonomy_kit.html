toolkit for process ncbi taxonomy tree data

The Taxonomy Database is a curated classification and nomenclature for all of the organisms in the public sequence databases. This currently represents about 10% of the described species of life on the planet.

.NET clr type export
taxonomy: Taxonomy

A simple organism taxonomy model.
taxonomy_node: TaxonomyNode

The tree node calculation model for NcbiTaxonomyTree, a labeled tree node for a specific ncbi taxid.
