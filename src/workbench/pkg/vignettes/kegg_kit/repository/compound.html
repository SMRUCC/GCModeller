compound {repository} R Documentation

construct a new kegg compound data model

Description

Usage

compound(entry, name, formula, exactMass,
    reaction = NULL,
    enzyme = NULL,
    remarks = NULL,
    KCF = NULL,
    DBLinks = NULL,
    pathway = NULL,
    modules = NULL);

Arguments

entry

the kegg compound id

name

the compound common names

formula

the chemical formula of the kegg compound

exactMass

the evaluated exact mass value based on the formula string

reaction

A set of the related kegg reaction data about current metabolite

enzyme

A set of the related enzyme that associated with the reaction list

remarks

comment text about this metabolite

KCF

the molecular strucutre text

DBLinks

A dataframe object that contains the external reference link to
other database of current metabolite object, data fields: "db", "id", "link" should
be exists in this dataframe object.
