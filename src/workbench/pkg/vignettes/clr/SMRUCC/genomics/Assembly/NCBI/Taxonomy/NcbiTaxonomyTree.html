NcbiTaxonomyTree

Description

Builds the following dictionnary from NCBI taxonomy ``nodes.dmp`` and ``names.dmp`` files ```json { Taxid : namedtuple('Node', ['name', 'rank', 'parent', 'children'] } ``` + https://www.biostars.org/p/13452/ + https://pythonhosted.org/ete2/tutorial/tutorial_ncbitaxonomy.html

Declare

            
# namespace SMRUCC.genomics.Assembly.NCBI.Taxonomy

   # ``{taxid -> taxonomy_node}``
   Taxonomy: list;
}
