Vessel

Description

一个反应容器，也是一个微环境，这在这个反应容器之中包含有所有的反应过程 (这是一个通用的生物化学反应过程模拟器，是整个虚拟细胞计算引擎的核心部件)

a container of a set of the reaction Channel and mass Factor.
