Array

Description

<Bag>、<Seq> 以及 <Alt> + <rdf:Bag> 元素用于描述一个规定为无序的值的列表。元素可包含重复的值。 + <rdf:Seq> 元素用于描述一个规定为有序的值的列表（比如一个字母顺序的排序）。 + <rdf:Alt> 元素用于一个可替换的值的列表（用户仅可选择这些值的其中之一）。
