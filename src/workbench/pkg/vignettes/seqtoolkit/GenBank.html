origin.fasta

get, add or replace the genome origin fasta sequence in the given genbank assembly file.

let origin.fasta as function(gb as  File, nt as  FastaSeq = <NULL>, mol_type as string = "genomic DNA") -> <generic> File|<generic> FastaSeq {
    #
    # .NET API information
    #
    # module: seqtoolkit.genbankKit
    # LibPath: D:/GCModeller/src/workbench/pkg/assembly/net6.0
    # library: seqtoolkit.dll
    # package: "GenBank"
    #
    return call R#.interop_[genbankKit::getOrAddNtOrigin](...);
}

Arguments

  • gb: -
  • nt: -
  • mol_type: -

Value

if the **`nt`** parameter is nothing,
means get fasta sequence, otherwise is add/update fasta
sequence in the genbank assembly, the returns type of
the api will change from the getted fasta sequence to
the modified genbank assembly object.
