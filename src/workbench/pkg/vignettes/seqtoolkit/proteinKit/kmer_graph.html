kmer_graph {proteinKit} R Documentation

Constructs k-mer adjacency graphs from protein sequence data. Nodes represent k-length

Description

subsequences, edges connect k-mers appearing consecutively in the sequence.

Usage

kmer_graph(prot,
    k = 3);

Arguments

prot

A FASTA sequence or collection of FASTA sequences to process.

k

The subsequence length parameter for k-mer generation. Default is 3. [as integer]

env

The R runtime environment for error handling and resource cleanup. [as Environment]

Details

Authors

seqtoolkit

Value

Returns a single KMerGraph for single sequence input. Returns a named list of KMerGraph objects for multiple sequences. Returns error message for invalid inputs.

clr value class

  • KMerGraph

Examples

 let graphs = kmer_graph(sequences, k = 3);
