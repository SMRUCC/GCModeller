as.proteins

Extract the protein sequences from the gene prediction result, and return as FASTA format. The sequence ID is in the format of "seqidgeneindex". For example, "contig15" means the 5th predicted gene on contig1. The sequence description is in the format of "start-end(strand)". For example, "100-900(+)" means the gene starts at position 100, ends at position 900, and is on the forward strand.
as.genes

Extract the gene sequences from the gene prediction result, and return as FASTA format. The sequence ID is in the format of "seqidgeneindex". For example, "contig15" means the 5th predicted gene on contig1. The sequence description is in the format of "start-end(strand)". For example, "100-900(+)" means the gene starts at position 100, ends at position 900, and is on the forward strand.
