get_xrefs {uniprot} R Documentation

get external database reference id set

Description

Usage

get_xrefs(prot,
    dbname = NULL);

Arguments

prot

target protein object to extract its database corss reference id. [as entry]

dbname

this function will returns a character vector of the db_xrefs for specific database if this db name is specificed. [as string]

Details

the uniprot database name will be named as: UniProtKB/Swiss-Prot for make unify with the genebank feature xrefs.

Authors

seqtoolkit

Value

this function returns data object in these one of the listed data types: dataframe, string.

clr value class

  • dataframe
  • string
