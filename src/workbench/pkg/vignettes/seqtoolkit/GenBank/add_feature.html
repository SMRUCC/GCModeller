add_feature {GenBank} R Documentation

add feature into a given genbank object

Description

Usage

add_feature(gb, feature);

Arguments

gb

[as File]

feature

[as Feature]

Details

Authors

seqtoolkit

Value

this function returns data object of type File.

clr value class

  • File

Examples

 let gb_asm = read.genbank("./xxx.gbff");
 let mics_site = GenBank::feature("micsRNA", nucl_location(5656,33,"+"));
 
 # use the operator
 gb_asm = gb_asm + mics_site;
 # or use the function
 gb_asm = gb_asm |> add_feature(mics_site);
