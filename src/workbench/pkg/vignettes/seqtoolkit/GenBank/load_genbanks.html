load_genbanks {GenBank} R Documentation

populate a list of genbank data objects from a given list of files or stream.

Description

Usage

load_genbanks(files,
    autoClose = TRUE);

Arguments

files

a list of files or file stream

autoClose

auto close of the Stream if the `files` contains stream object?. [as boolean]

env

[as Environment]

Details

this function supports of read assembly data directly from *.gz genbank archive file.

Authors

seqtoolkit

Value

this function returns data object of type File.
