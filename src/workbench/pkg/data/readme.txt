## kegg data

+ kegg.zip

a kegg reference database file which is generated via 
the ``kegg_api`` package. the file strucutres of this
database file is:

    + <pathway>/
       + map.xml
       
       + compounds/
       + modules/

            + <module>
               + module.xml
               
               + reactions/
               + compounds/
