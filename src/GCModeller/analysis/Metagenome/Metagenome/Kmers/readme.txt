数据库文件说明

   文件夹结构        文件描述说明
   ---------        ------------

+- seq_ids.csv      一个用于存储来源序列信息的Excel表格文件，可以通过Excel程序打开自由编辑序列名称信息或者序列对应的ncbi物种id信息，但是序列的id信息请勿进行变更
+- config.txt       进行数据库构建所使用的参数信息，参数信息使用json格式进行存储
|
+- data             kmer数据库文件夹
   |   
   +- ...           按照kmer片段前缀进行数据库切片的划分
      |
	  +- index.dat  切片的数据库文件索引 
	  +- kmers.dat  切片的数据库文件，存储着kmer序列片段在不同的物种来源的序列数据上的分布信息，序列来源信息存储于seq_ids.csv表格文件中
	  
	  
请注意：
为了避免数据库损坏，请不要直接用文本编辑器编辑data文件夹下的任何数据库文件，尽量避免使用文本编辑器打开任何以*.dat后缀结尾的数据库文件
